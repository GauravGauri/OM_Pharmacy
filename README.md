## ✨ Key Features & User Experience Highlights

OM Pharmacy stands out with its premium user interface and fluid micro-interactions:

*   **✨ Interactive Fluid Custom Cursor**: A dual-element custom cursor (pointer + trailing ring) that smooth-tracks mouse movements for desktop users.
*   **🧩 Intersection Observer Animations**: Smooth fade-in, slide-left, and slide-right entry animations trigger dynamically as the user scrolls down the page.
*   **📊 Dynamic Animated Statistics Counter**: A custom JavaScript-powered counter that counts up to the target number (e.g., `5000+` Medicines) when the stats section scrolls into view.
*   **💬 WhatsApp Floating Action Button**: A persistent floating button that lets users connect with the pharmacist via WhatsApp with a single click.
*   **📥 Simulated Dynamic Form Submission**: The contact form features a realistic send state, displaying a loading spinner during submission and transitioning into a green success checkmark upon completion.
*   **📱 Fully Responsive Navigation**: A collapsible hamburger menu with smooth toggle actions, optimised for devices of all screen sizes.
*   **📍 Localized Map Integration**: Embedded Google Maps location showing the physical store address in New Tehri, Uttarakhand.
*   **🔄 Scrolling Medicine Marquee (Ticker Track)**: An infinite looping horizontal scrolling bar highlighting key categories such as *Prescription Medicines*, *Ayurvedic Products*, *Veterinary Care*, and *Pediatric Medicines*.

## 🛠️ Installation & Usage

Since this is a lightweight, frontend-only project, no compilation or build step is required! You can run it locally in seconds:

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/GauravGauri/OM_Pharmacy.git
    cd OM_Pharmacy
    ```
2.  **Open the Web App**
    *   Simply double-click the `index.html` file to open it in your web browser.
    *   *Or* run a simple local development server. For example, if you have VS Code, you can use the **Live Server** extension, or run:
        ```bash
        # Using Python
        python -m http.server 8000
        
        # Using Node.js (npx)
        npx serve
        ```
    *   Open `http://localhost:8000` (or the port provided) in your browser.
